# components/
> L2 | Parent: /CLAUDE.md

Shared UI components consumed by screens and features.

## Members

- `card/`: Pure presentational card rendering — CardField + CardDisplay (Tailwind-styled, no store access)
- `cards/`: BusinessCard template renderers
- `modals/`: Sheet modals (ShareMenu, BackgroundPicker, AddBlock, etc.)
- `qr/`: QR code + LinkedIn-styled QR (StyleSheet-based)
- `shared/`: Tailwind-styled shared primitives — AdaptiveGlass, Avatar, QRCode
- `ui/`: Avatar, AnimatedComponents (StyleSheet-based originals)
