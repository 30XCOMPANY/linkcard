# tw/
> L2 | Parent: src/CLAUDE.md

CSS wrapper layer for React Native components — bridges `react-native-css` `useCssElement` with standard RN/Expo primitives so consumers can use `className` props (Tailwind).

## Members

- `index.tsx`: Core wrappers — View, Text, ScrollView, Pressable, TextInput, Link, AnimatedScrollView, TouchableHighlight, useCSSVariable
- `image.tsx`: Image wrapper — remaps CSS objectFit/objectPosition to expo-image contentFit/contentPosition
- `animated.tsx`: Animated wrapper — extends Reanimated with CSS-wrapped View
